# Advanced GitHub Slug Action

This is an advanced version of the GitHub Slug Action with the following features:

- AI-powered slug generation for improved readability and uniqueness.
- Multi-language and localization support.
- Plugin system for custom slug transformations.
- CLI tool for previewing slug generation.
- Security enhancements and input sanitization.
- Analytics and telemetry support.
- Enhanced CI/CD workflows.

## Usage

Replace the slugify-value steps in your workflow with this action's steps.

## CLI Tool

Use the CLI tool to preview slug generation:

```bash
sluggen --value "Your string here" --ai --maxLength 50
```
